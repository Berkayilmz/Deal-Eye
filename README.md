# **Yapım Aşamasındadır / Work in Progress**  

## Proje Tanıtımı: Deal-Eye  

**Deal-Eye**, alışveriş alışkanlıklarınıza teknolojiyle yenilikçi bir dokunuş katmak için geliştirildi.  
Python ve Selenium kullanarak, **Şok Market**'teki tüm ürünlerin resimlerini, başlıklarını ve fiyat bilgilerini toplayan bir sistem oluşturduk. Bu veriler Firebase üzerinde saklanıyor ve React Native ile geliştirilmiş mobil uygulamamızda kullanıcıların erişimine sunuluyor.  

### Uygulama Özellikleri  

- **Ürün Görüntüleme**:  
  Kullanıcılar, **Deal-Eye** ile geniş ürün yelpazesini kolayca keşfedebilir. Şok Market'teki ürünlerin tüm detayları (resim, başlık ve fiyat) kullanıcı dostu bir arayüzde sunulmaktadır.  

- **Fiyat Analizi ve Tahmin**:  
  Her ürünün detay sayfasında:  
  - **Geçmiş 1 haftalık fiyat bilgisi grafiği** ile fiyat trendlerini takip edebilirsiniz.  
  - **Yarınki tahmini fiyat bilgisi**, alışveriş kararlarınızda size rehberlik eder.  

### Teknolojiler  

- **Backend**: Python & Selenium ile veri çekme ve Firebase entegrasyonu  
- **Frontend**: React Native ile kullanıcı dostu mobil uygulama  
- **Veri Analizi**: Fiyat tahmini için gelişmiş algoritmalar  

### Hedefimiz  

**Deal-Eye**, kullanıcıların alışveriş sürecini hem şeffaf hem de keyifli bir hale getirmek için tasarlandı. Gelişmiş veri analizi ve tahmin algoritmalarımız sayesinde kullanıcılarımız her zaman bir adım önde olacak.  

Deal-Eye dünyasına hoş geldiniz!

## **Work in Progress**  

## Project Overview: Deal-Eye  

**Deal-Eye** is developed to add an innovative touch to your shopping habits with the power of technology.  
Using Python and Selenium, we have built a system to collect all product images, titles, and price details from **Şok Market**. These data are stored in Firebase and made accessible to users via our mobile application developed with React Native.  

### Application Features  

- **Product Display**:  
  Users can easily explore a wide range of products with **Deal-Eye**. All product details (image, title, and price) from Şok Market are presented in a user-friendly interface.  

- **Price Analysis and Prediction**:  
  On each product’s detail page:  
  - **A 1-week price history graph** lets you track price trends.  
  - **Predicted price for tomorrow** helps guide your shopping decisions.  

### Technologies  

- **Backend**: Data scraping with Python & Selenium, integrated with Firebase  
- **Frontend**: User-friendly mobile application built with React Native  
- **Data Analysis**: Advanced algorithms for price prediction  

### Our Goal  

**Deal-Eye** is designed to make the shopping process both transparent and enjoyable. With advanced data analysis and prediction algorithms, our users will always stay one step ahead.  

Welcome to the world of Deal-Eye!
